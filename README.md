# Deployment
Requires a .htaccess file with the following format for local deployment:
```
SetEnv DBUSERNAME "******************"
SetEnv DBPASSWORD "******************"
SetEnv DBNAME "******************"
SetEnv DBHOST "******************"
SetEnv DBPORT "******************"
```
