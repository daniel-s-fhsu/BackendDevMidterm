# Deployment
Requires a .htaccess file with the following format for local deployment:
```
SetEnv USERNAME "******************"
SetEnv PASSWORD "******************"
SetEnv DBNAME "******************"
SetEnv HOST "******************"
SetEnv PORT "******************"
```
